# 学習用AIエージェント セットアップガイド

このガイドに従って設定すると、プログラミング学習をサポートしてくれるAIエージェントが使えるようになります。

---

## 1. Node.js のインストール

Claude Code を動かすために、まず Node.js が必要です。

1. https://nodejs.org/ にアクセス
2. **LTS版**（左側のボタン）をダウンロード
3. インストーラーを実行（すべてデフォルトでOK）
4. 確認: ターミナルを開いて以下を入力
   ```
   node --version
   ```
   → `v20.x.x` のようにバージョンが表示されればOK

### ターミナルの開き方
- **Mac**: Spotlight（Cmd + Space）→「ターミナル」と入力
- **Windows**: スタートメニュー →「PowerShell」と入力

---

## 2. Claude Code のインストール

ターミナルで以下のコマンドを実行:

```
npm install -g @anthropic-ai/claude-code
```

完了したら確認:
```
claude --version
```
→ バージョンが表示されればOK

---

## 3. アカウント認証

1. ターミナルで `claude` と入力して Enter
2. ブラウザが開くので、Anthropic アカウントでログイン
3. 認証が完了するとターミナルに戻ります
4. `/exit` で一旦終了

---

## 4. 学習キットの設置

配布されたファイルを、自分の学習用フォルダにコピーします。

### フォルダ構成
```
あなたの学習フォルダ/
├── CLAUDE.md          ← ルール概要（AIの役割を定義）
└── .claude/
    └── rules/         ← 詳細なルール（常に有効）
        ├── teaching.md
        ├── hint.md
        ├── documentation.md
        └── progress.md
```

### 手順

1. 学習用のフォルダを作る
   ```
   mkdir my-learning
   cd my-learning
   ```

2. 配布された learning-kit の中身をすべてコピーする
   - `CLAUDE.md` → フォルダ直下に置く
   - `.claude/` フォルダ → フォルダ直下に置く

3. 確認: フォルダの中身を見る
   ```
   ls -la
   ```
   → `CLAUDE.md` と `.claude` が見えればOK

---

## 5. 使ってみよう

学習フォルダで Claude Code を起動:

```
cd my-learning
claude
```

### 基本操作

| 操作 | やり方 |
|------|--------|
| 質問する | テキストを打って Enter |
| ファイルを見せる | 「index.html を読んで」 |
| 終了する | `/exit` または Ctrl+C |

### 使い方の例

ルールは自動で有効になるので、特別なコマンドは不要です。普通に話しかけるだけでOK。

#### コードの解説を頼む
```
index.html のコードを解説して
```
→ 1行ずつ解説し、公式ドキュメントのURLも添えてくれます。

#### エラーを相談する
```
エラーが出て動かない
```
→ 答えをすぐ書かず、3段階のヒントで自分で考えられるように導いてくれます。
→ 「もっとヒント」と言えば次の段階のヒントがもらえます。

#### 課題に取り組む
```
じゃんけんゲームを作りたい
```
→ 小さなステップに分解して、1つずつ進めてくれます。
→ 進捗管理用のMDファイルも自動で作成されます。

## ルールの説明

### ルール = AIが常に守る行動指針

`.claude/rules/` フォルダに入っているファイルが、AIの行動ルールです。
Claude Code を起動するたびに自動で読み込まれます。

| ファイル | 内容 |
|---------|------|
| `teaching.md` | 日本語で回答、考え方→ヒント→コードの順で教える |
| `hint.md` | 答えを書かず3段階のヒントで導く |
| `documentation.md` | 公式ドキュメントのURLを根拠に説明する |
| `progress.md` | 課題の進捗をMDファイルで管理する |

### CLAUDE.md = ルールの概要
プロジェクト直下にある `CLAUDE.md` は、AIの役割（プログラミングの先生）を定義しています。
詳細なルールは `.claude/rules/` に分割されています。

## 困ったとき

### 「claude: command not found」と出る
→ Node.js のインストールからやり直してください

### 認証画面が出ない
→ ブラウザを手動で開いて claude.ai にアクセスしてください

### ルールが効いていない気がする
→ `.claude/rules/` フォルダが正しい場所にあるか確認してください
→ フォルダ構成が上記の通りになっているか確認
