# 学習用AIエージェント セットアップガイド

このガイドに従って設定すると、プログラミング学習をサポートしてくれるAIエージェントが使えるようになります。

---

## 1. Node.js のインストール

Claude Code を動かすために、まず Node.js が必要です。

1. https://nodejs.org/ にアクセス
2. **LTS版**（左側のボタン）をダウンロード
3. インストーラーを実行（すべてデフォルトでOK）
4. 確認: ターミナルを開いて以下を入力
   ```
   node --version
   ```
   → `v20.x.x` のようにバージョンが表示されればOK

### ターミナルの開き方
- **Mac**: Spotlight（Cmd + Space）→「ターミナル」と入力
- **Windows**: スタートメニュー →「PowerShell」と入力

---

## 2. Claude Code のインストール

ターミナルで以下のコマンドを実行:

```
npm install -g @anthropic-ai/claude-code
```

完了したら確認:
```
claude --version
```
→ バージョンが表示されればOK

---

## 3. アカウント認証

1. ターミナルで `claude` と入力して Enter
2. ブラウザが開くので、Anthropic アカウントでログイン
3. 認証が完了するとターミナルに戻ります
4. `/exit` で一旦終了

---

## 4. 学習キットの設置

配布されたファイルを、自分の学習用フォルダにコピーします。

### フォルダ構成
```
あなたの学習フォルダ/
├── CLAUDE.md          ← ルール（AIの動き方を決める）
└── .claude/
    └── skills/        ← スキル（AIに実行させる操作）
        ├── explain/
        │   └── SKILL.md
        ├── hint/
        │   └── SKILL.md
        └── step-by-step/
            └── SKILL.md
```

### 手順

1. 学習用のフォルダを作る
   ```
   mkdir my-learning
   cd my-learning
   ```

2. 配布された learning-kit の中身をすべてコピーする
   - `CLAUDE.md` → フォルダ直下に置く
   - `.claude/` フォルダ → フォルダ直下に置く

3. 確認: フォルダの中身を見る
   ```
   ls -la
   ```
   → `CLAUDE.md` と `.claude` が見えればOK

---

## 5. 使ってみよう

学習フォルダで Claude Code を起動:

```
cd my-learning
claude
```

### 基本操作

| 操作 | やり方 |
|------|--------|
| 質問する | テキストを打って Enter |
| ファイルを見せる | 「index.html を読んで」 |
| 終了する | `/exit` または Ctrl+C |

### スキルを使う

3つのスキルが使えます:

#### `/explain` - コード解説
コードの意味を1行ずつ解説してくれます。
```
/explain index.html
```
```
/explain この関数を解説して
```

#### `/hint` - ヒント
答えを教えずに、考え方のヒントだけくれます。
```
/hint エラーが出て動かない
```
```
/hint for文の書き方が分からない
```

#### `/step-by-step` - ステップバイステップ
やりたいことを小さなステップに分けて、1つずつ進めてくれます。
```
/step-by-step じゃんけんゲームを作りたい
```
```
/step-by-step HTMLで自己紹介ページを作りたい
```

## ファイルの説明

### CLAUDE.md（ルール）
- AIが**常に守る**ルール
- 「日本語で答える」「いきなり答えを書かない」などの基本ルールが書いてある
- 学校でいう**校則**のようなもの
- Claude Code を起動するたびに自動で読み込まれる

### SKILL.md（スキル）
- `/コマンド名` で**呼んだときだけ動く**操作
- 「コードを解説する」「ヒントを出す」など、特定の操作が定義してある
- 学校でいう**授業**のようなもの
- 自分で呼ばないと動かない

### 違いのまとめ

| | ルール（CLAUDE.md） | スキル（SKILL.md） |
|--|-------------------|-------------------|
| いつ動く | **常に** | **呼んだときだけ** |
| 例え | 校則 | 授業 |
| 設定場所 | プロジェクト直下 | .claude/skills/スキル名/ |
| 役割 | AIの基本的な振る舞い | 特定の操作の手順 |

## 困ったとき

### 「claude: command not found」と出る
→ Node.js のインストールからやり直してください

### 認証画面が出ない
→ ブラウザを手動で開いて claude.ai にアクセスしてください

### スキルが動かない
→ `.claude/skills/` フォルダが正しい場所にあるか確認してください
→ フォルダ構成が上記の通りになっているか確認

### その他
→ 講師に聞いてください
